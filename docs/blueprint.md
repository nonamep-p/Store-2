# **App Name**: Zenith Market

## Core Features:

- Product Catalog: Browse and view available products with details and images.
- Shopping Cart: Add products to a shopping cart for purchase.
- Checkout Process: Simple and secure checkout process with payment gateway integration.
- AI Product Recommendations: AI tool that suggests related items based on viewed or carted products.
- User Accounts: User account creation and management for order history and saved information.
- Product Search: Search functionality to quickly find products.
- Product Detail Page: Page to display the product.
- Quick View: Quickly view product details without navigating to a separate page.
- Filters, Categories, Tags: Allows users to filter and sort product listings by category and tags

## Style Guidelines:

- Primary color: Soft Blue (#77B5FE) to evoke trust and tranquility.
- Background color: Very Light Blue (#EBF4FA), for a calm, unobtrusive feel.
- Accent color: Pale Yellow (#FAF4DC) to highlight key actions.
- Body and headline font: 'Inter' sans-serif for a modern, clean aesthetic.
- Use simple, minimalist icons for navigation and product features.
- Clean, grid-based layout with plenty of white space to highlight products.
- Subtle transitions and loading animations to enhance user experience.
